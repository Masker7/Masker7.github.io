绘制类图

安装

插件

  • PlantUML Integration
  • PlantUML Parser

下载

在 IDEA -> Settings -> Plugins -> Marketplace 搜索即可

配件

  • graphviz-dot

PlantUML Parser 依赖 PlantUML 插件 (PlantUML 不仅仅是一个插件，官网还提供了一个独立的 Java 程序（jar 包），我们还可以通过这个独立程序来绘制 UML 图)，PlantUML 依赖 graphviz-dot 软件包，所以，我们首先需要安装 graphviz-dot。
