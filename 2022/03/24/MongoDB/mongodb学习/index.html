2. 删除

  • 参数说明

  • query： 可选删除的文档的条件

  • justOne： 如果设为 true 或 1 代表只删除一个文档；如果不设置该参数，默认为 false，表示删除所有匹配条件的文档

  • writeConcern： 可选抛出异常的级别

3. 更改

参数说明

  • query： 条件，类似 sql 中 update 语句后的 where
  • update： 更新内容，类似 sql 中 update 语句后的 set
  • upsert： 可选，如果不存在 update 的记录，是否插入 objNew，true 为插入，默认为 false
  • multi： 可选，默认是 false， 只更新找到的第一条记录；如果 true，则更新所有匹配条件的值
  • writeConcern： 可选，抛出异常的级别
